-----------------------------------
「Screening」 ver3.00 (2018/10/30)
-----------------------------------

■使い方
  ※必ずRSSとログインしたMarketSpeedを起動しておいてください。
  プログラムが正常に終了すると、Excelファイルが閉じます。
  
  ★前日スクリーニング.xlsm★
     15時を過ぎたら実行してください。
     [前日スクリーニング.xlsm]を起動して、[スタート]マクロを実行するとプログラムが実行されます。
     プログラムが実行されると[☆前日スクリーニング]フォルダに、
     HYPER SBIにインポート出来る形式の[買い-1.csv]、[買い-2.csv]、[売り-2.csv]が作成されます。
    
    ◆所要時間
       5分〜10分程

■ファイル構成
   当日スクリーニング.xlsm  当日のスクリーニング
   前日スクリーニング.xlsm  前日のスクリーニング
   readme.txt               このファイル
   [Plugin]                 プログラムの実行に必要なファイルが入っているフォルダ
   [☆当日スクリーニング]     当日スクリーニング実行後に作成されるファイルが入るフォルダ
   [☆前日スクリーニング]     前日スクリーニング実行後に作成されるファイルが入るフォルダ
